# Landing APP

Landing challenge.

![image](https://i.imgur.com/RTnSm3T.png)

### Requirements

To run this project locally you should have all these requirements.

- node.js v14.18.1
- yarn v1.22.15

### RUN PROJECT

> Serve the project

```
cd vanilla-landing
npx servor .
```

Then you can put the dist folder on a server to serve the web app :)
